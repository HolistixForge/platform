# frontend-data

Shared data layer for the Holistix Forge frontend.

## Documentation

- [Frontend Architecture](../../doc/architecture/FRONTEND_ARCHITECTURE.md)

This package provides:

- `ApiContext` for application-wide API access
- `GanymedeApi` client with OAuth token + gateway support
- React Query hooks for users, projects, permissions, and gateway data
- Storybook helpers for mocking API interactions
